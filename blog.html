The hardest thing I had to work through for this assignment was getting my page hosted through GitHub pages. I wanted to be able to keep my websites after I graduated Oregon State, so I decided to host my websites on GitHub pages. This posed extra challenges due to the security permissions on the class repository for the assignment. Thankfully, the professor for the class sent us a link to have our GitHub pages published.

I already knew the basic structure of how websites are coded (html, CSS, and JS), but I had no experience actually using them. To help me get my bearings with html, I used several online resources as well as the lectures. The online resource I used the most was W3 school's article on html.
